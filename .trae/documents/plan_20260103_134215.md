# LangChain & LangGraph Agent 实现计划

## 项目结构

在 `sandbox/examples/` 目录下创建两个新项目：

### 1. LangChain Agent (`langchain-agent`)

基于 LangChain 的 ReAct Agent，集成 agent-sandbox 工具

### 2. LangGraph Agent (`langgraph-agent`)

基于 LangGraph 工作流的 Agent，支持多步骤推理

## 技术架构

### 共享配置

* **沙箱**: agent-infra (本地 Docker: `http://localhost:8080`)

* **模型**: 火山云 Volcengine API (DeepSeek-V3)

* **依赖**: agent-sandbox SDK + LangChain/LangGraph

### 环境变量

```bash
# 火山云配置
COZE_WORKLOAD_IDENTITY_API_KEY=your_api_key
COZE_INTEGRATION_MODEL_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
CODE_LL_MODEL=deepseek-v3-2-251201

# 沙箱配置
SANDBOX_BASE_URL=http://localhost:8080
```
